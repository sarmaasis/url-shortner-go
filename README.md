# url-shortner-go
Demonstration of URL shortner with API Rate Limiter and Custom URL using Redis.



### Built With

* Go
* Redis
* Docker
* GoFiber

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Prerequisites

* Go
* Docker Compose

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sarmaasis/url-shortner-go
   ```
2. Change Directory
   ```sh
   cd url-shortner-go
   ```
3. 
   ```sh
   docker-compose up
   ```
