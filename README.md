### Prerequisites

* Go
* Docker-Compose

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sarmaasis/url-shortner-go
   ```
2.```sh
   cd ur-shortner-go
   ```
3. 
   ```sh
   docker-compose up
   ```
